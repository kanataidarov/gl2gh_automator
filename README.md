
# GitLab to GitHub Migrator

## Features
- Clone GitLab repository using Personal Access Token
- Push to GitHub repository
- Migrate issues and merge requests via API

## Usage
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Edit `main.py` and add your tokens and repo URLs.
3. Run the script:
   ```bash
   python main.py
   ```
